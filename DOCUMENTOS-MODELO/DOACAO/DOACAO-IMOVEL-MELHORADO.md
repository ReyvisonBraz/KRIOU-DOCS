# CONTRATO DE DOAÇÃO DE IMÓVEL

## 📋 Descrição
Modelo de contrato de doação de imóvel (casa, apartamento, terreno). A doação é o ato pelo qual uma pessoa transfere gratuitamente bens de seu patrimônio para o de outra. Este modelo inclui cláusulas de blindagem.

## ⚖️ Legislação Aplicável
- Código Civil Brasileiro, Arts. 538 a 564

### 8.2. Ausência de Vícios
