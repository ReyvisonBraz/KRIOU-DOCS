## ✏️ Como preencher os campos

### Campos para preenchimento

Os campos para preencher são indicados assim:

```
📝 Nome completo: ________________________________________________
```

**O que fazer:** Escreva sua resposta no espaço disponível.

### Campos com opções

Quando há opções, você escolhe marcando com **X**:

```
[ ] Apartamento
[ ] Casa
[X] Sobrado
```

**O que fazer:** Coloque um **X** apenas na opção que se aplica ao seu caso.

### Campos com valores

Quando há valores monetários ou numéricos:

```
**Valor do aluguel:** R$ _______________
```

**O que fazer:** Escreva o valor em números no espaço.

## 📅 Formato de datas

### Formato correto

Seguimos o formato brasileiro padrão:

```
dia de mês por extenso de ano
```

**Exemplos:**

| Correto | Incorreto |
|---------|-----------|
| 12 de abril de 2026 | 12/04/2026 |
| 1 de janeiro de 2025 | 01/01/2025 |
| 25 de março de 2026 | 25/03/26 |

### Regras:

1. **Dia:** Sempre em números (12, 25, 1)
2. **Mês:** Sempre por extenso (janeiro, fevereiro, março...)
3. **Ano:** Sempre em números (2025, 2026)

### Lista de meses:

```
janeiro, fevereiro, março, abril, maio, junho
julho, agosto, setembro, outubro, novembro, dezembro
```

## ✍️ Assinaturas e testemunhas

### Onde colocar as assinaturas

Os contratos devem ser assinados:

1. **Pelo contratante** – no campo indicado
2. **Por 2 testemunhas** – sempre presentes

### Modelo de assinatura:

```
________________________________________________
Nome: João da Silva
CPF: 123.456.789-00
```

### Regras para testemunhas:

✅ Devem ser maiores de 18 anos

✅ Devem ter capacidade civil

✅ Devem conhecer pelo menos uma das partes

✅ Devem presenceciar a assinatura

### Local e Data

Sempre preencha o local (cidade) e a data:

```
**Cidade:** São Paulo

**Data:** 12 de abril de 2026
```

## 🔤 Caracteres especiais

### Quando usar

Caracteres especiais são usados em:

- Numeração de cláusulas (§, 1°, 2°)
- Ordinais (1º, 2º, 3º)
- Plural (ões, ão)
- Nomes especiais

### Tabela de caracteres comuns:

| Para conseguir | Use |
|----------------|-----|
| Parágrafo | § |
| Grau | º (Ctrl + , + o) |
| Fem. Grau | ª (Ctrl + , + a) |
| Cedilha | Ç (Ctrl + , + c) |
| Til | ã (Alt + 227) |
| Acento agudo | é (Ctrl + ' + e) |

### Como digitar no Windows:

1. **Parágrafo (§):** Alt + 21
2. **Grau (º):** Ctrl + Shift + X, ou Alt + 167
3. **Cedilha (Ç):** Ctrl + , + c

### Exemplo de uso:

```
CLÁUSULA PRIMEIRA – DO OBJETO

§ 1º O primeiro parágrafo...

§ 2º O segundo parágrafo...

1º - Primeiro item
2º - Segundo item
```

## 🖨️ Impressão e formatação

### Configuração recomendada

Para impressão de contratos:

1. **Papel:** A4 (210 x 297 mm)
2. **Margens:**
   - Superior: 2,5 cm
   - Inferior: 2,5 cm
   - Esquerda: 3,0 cm
   - Direita: 2,0 cm

3. **Fonte:** Times New Roman ou Arial
4. **Tamanho:** 12 pt para texto normal
5. **Alinhamento:** Justificado

### Dicas de impressão:

✅ Imprima em papel branco, sem sulfite

✅ Use tinta preta

✅ Verifique se a impressora está com toner suficiente

✅ Faça uma cópia antes de levar ao cartório

## ❓ Dúvidas frequentes

### Posso modificar os modelos?

R: Sim, os modelos são apenas referência. Mas tome cuidado para não alterar cláusulas importantes.

### Preciso de advogado?

R: Para contratos simples, não é obrigatório. Mas é recomendável para casos complexos.

### As testemunhas precisam ter documentos?

R: Sim, anote o nome completo, CPF e RG das testemunhas.

### Onde registrar o contrato?

R: Depende do tipo:
- Imóveis: Cartório de Registro de Imóveis
- Outros: Cartório de Notas

### Quanto custa para reconhecer firma?

R: Varia por Estado e cartório. Consulte o site da OAB local.
