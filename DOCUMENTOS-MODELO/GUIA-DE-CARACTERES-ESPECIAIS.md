## 🔤 Caracteres Especiais

### Parágrafo (§)

**Uso:** Indica parágrafos dentro de cláusulas

**Como digitar:**
- Windows: Alt + 21
- Mac: Option + 6
- Linux: Alt Gr + 6

**Exemplo:**
```
§ 1º O primeiro parágrafo desta cláusula...
§ 2º O segundo parágrafo estabelece que...
```

### Ordinal Masculino (º)

**Uso:** 1º, 2º, 3º (primeiro, segundo, terceiro)

**Como digitar:**
- Windows: Ctrl + Shift + X, depois letra
- Ou: Alt + 167

**Exemplo:**
```
1º (primeiro)
2º (segundo)
3º (terceiro)
```

### Ordinal Feminino (ª)

**Uso:** 1ª, 2ª, 3ª (primeira, segunda, terceira)

**Como digitar:**
- Windows: Ctrl + Shift + X, depois letra
- Ou: Alt + 166

**Exemplo:**
```
1ª (primeira)
2ª (segunda)
3ª (terceira)
```

### Cedilha (Ç)

**Uso:** Nomes e palavras com Ç

**Como digitar:**
- Windows: Ctrl + , + c
- Mac: Option + c
- Linux: Alt Gr + , + c

**Exemplo:**
```
São Paulo
Caçador
çuíto
```

## 📅 Formatação de Datas

### Formato Padrão Brasileiro

```
[dia] de [mês por extenso] de [ano]
```

**Exemplos:**

| Data | Formato Correto |
|------|-----------------|
| 12/04/2026 | 12 de abril de 2026 |
| 01/01/2025 | 1 de janeiro de 2025 |
| 25/12/2024 | 25 de dezembro de 2024 |

### Meses do Ano (por extenso)

| Número | Mês |
|--------|-----|
| 01 | janeiro |
| 02 | fevereiro |
| 03 | março |
| 04 | abril |
| 05 | maio |
| 06 | junho |
| 07 | julho |
| 08 | agosto |
| 09 | setembro |
| 10 | outubro |
| 11 | novembro |
| 12 | dezembro |

### Importante:

❌ Não use: 12/04/2026

❌ Não use: 12 de Abr de 2026

✅ Use: 12 de abril de 2026

## 📝 Numeração de Cláusulas

### Cláusulas Principais

```
CLÁUSULA PRIMEIRA – DO OBJETO
CLÁUSULA SEGUNDA – DO PRAZO
CLÁUSULA TERCEIRA – DAS PENALIDADES
```

### Subcláusulas e Parágrafos

```
§ 1º - Primeiro parágrafo...
§ 2º - Segundo parágrafo...

§ 1º O primeiro parágrafo...
§ 2º O segundo parágrafo...
```

### Incisos (Romanos)

```
I – primeiro inciso
II – segundo inciso
III – terceiro inciso
IV – quarto inciso
V – quinto inciso
```

### Alineas (Letras)

```
a) primeira alínea
b) segunda alínea
c) terceira alínea
```

### Exemplo Completo:

```
CLÁUSULA TERCEIRA – DAS PENALIDADES

§ 1º O atraso no pagamento sujeitará o devedor a...

§ 2º Em caso de inadimplemento, aplicar-se-ão as seguintes penalidades:

I – multa de 2% sobre o valor;

II – juros de 1% ao mês;

III – correção monetária pelo IPCA.

a) As penalidades são cumulativas;

b) Não excluem outras cominações legais.
```

## 📐 Formatação de Texto

### Negrito

**Uso:** Destacar informações importantes

**Como aplicar:**
- Selecione o texto e pressione Ctrl + B
- Ou use **texto** no markdown

**Exemplo:**
```
**ATENÇÃO:** Este campo é obrigatório
```

### Itálico

**Uso:** Termos em latim ou ênfase

**Como aplicar:**
- Selecione o texto e pressione Ctrl + I
- Ou use *texto* no markdown

**Exemplo:**
```
*mutatis mutandis*
```

### Sublinhado

**Uso:** Raramente usado em documentos jurídicos

**Como aplicar:**
- Selecione o texto e pressione Ctrl + U

## ✍️ Exemplos Práticos

### Exemplo 1: Cláusula com parágrafos

```
CLÁUSULA TERCEIRA – DO PREÇO

O preço total do imóvel é de R$ 500.000,00 (quinhentos mil reais).

§ 1º O pagamento será realizado em 10 (dez) parcelas mensais.

§ 2º A primeira parcela vencerá em 1º de janeiro de 2026.

§ 3º O não pagamento até o vencimento sujeitará o devedor a:
I – multa de 2%;
II – juros de 1% ao mês;
III – correção monetária.
```

### Exemplo 2: Data correta

```
São Paulo, 15 de abril de 2026.
```

❌ Errado: 15/04/2026

❌ Errado: 15 de Abr de 2026

✅ Correto: 15 de abril de 2026

### Exemplo 3: Lista com alíneas

```
As partes concordam com as seguintes condições:

a) prazo de 12 (doze) meses;

b) valor de R$ 1.000,00 (mil reais);

c) multa de 10% em caso de inadimplemento.
```

## 📌 Dicas de Formatação

|--------|--------|
| Superior | 2,5 cm |

| Esquerda | 3,0 cm |

|------|-------|---------|
| Texto normal | Times New Roman | 12 pt |

| Dados de identificação | Times New Roman | 11 pt |

| Elemento | Alinhamento |
|----------|-------------|
| Texto normal | Justificado |
| Endereços | Esquerdo |
| Datas | Esquerdo |
