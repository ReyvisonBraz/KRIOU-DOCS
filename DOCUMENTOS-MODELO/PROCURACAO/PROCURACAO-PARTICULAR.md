# PROCURAÇÃO PARTICULAR

## Descrição
Modelo de procuração particular para constituição de representante em negócios ou atos jurídicos diversos. Este modelo serve para delegar poderes gerais para representação em procedimentos administrativos, atendimento ao consumidor, etc.

## Legislação Aplicável
- Código Civil Brasileiro, Arts. 105 a 119

## Observações

- **Campos obrigatórios:** Nome completo do outorgante, CPF do outorgante, RG do outorgante, endereço do outorgante, nome completo do outorgado, CPF do outorgado, poderes conferidos, data, assinatura do outorgante
- **Campos opcionais:** Telefone, e-mail, prazo de validade
- **Variantes identificadas:** Procuração ad juditia (poderes judiciais), procuração para Receita Federal, procuração para órgãos públicos, procuração específica para bancos
- **Validade:** Sem prazo definido, válido enquanto não revokedo
- **Substabelecimento:** Deve ser expressamente autorizado se permitido
- **Fonte:** PROCON-SP (procon.sp.gov.br)
- **Data de coleta:** Abril 2026
